# Ekstra AI Router - User Journey

## The Problem We're Solving

Vending machine operators using TP-Link AX1500 routers face a **30-60 minute manual setup** with a **20-step process** that frequently fails. When issues occur (captive portals, signal drops, subnet conflicts), operators have no visibility and must make support calls or on-site visits.

## The Ekstra Solution

A smart router that **sets up in 10 minutes** and **heals itself** when issues occur.

## What Happens Behind the Scenes

While the operator completes these simple steps, the Ekstra firmware is:

### During Setup
- Detecting venue network subnet to avoid conflicts
- Selecting optimal WiFi channel
- Configuring DHCP with safe range
- Registering with api.ekstra.ai
- Starting monitor mode for analytics

### After Setup (Automatic)
- Monitoring connection health every 30 seconds
- Auto-reconnecting if WiFi drops
- Re-authenticating captive portals when sessions expire
- Uploading telemetry to cloud every 5 minutes
- Collecting anonymized location analytics
- Checking for OTA updates
